# To-Do List

This is a simple to-do list application.

### Features
- Add tasks
- Mark tasks as completed
- Delete tasks

### Usage
- Add your tasks in the input field and click "Add".
- Mark tasks as completed by clicking the checkbox.
- Delete tasks by clicking the delete button.
